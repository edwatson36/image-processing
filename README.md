# image-processing
Repo of e2e scripts and source code for image based ML use cases.
Notebooks developed in Google Colab.
See README.txt files for packages in src/ folder for testing status.

# Usage
Clone this repository.
Upload/prepare your data in the data/ folder.
Use scripts in the src/ folder or explore workflows in notebooks/.

# Folder Structure
src/           # Source code for reusable snippets and utilities
notebooks/     # Example e2e implementations of functions and scripts dealing with image based use cases
data/          # Raw and processed datasets
docs/          # Documentation
